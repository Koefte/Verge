# Verge

## Commands

-npm run build : builds the project
-npm run start : runs index
-npm run serve : serves frontend on localhost
-npm run dev   : tsc in watch mode
-npm run test  : check test cases

## About

This project aims to create a convergence test machine for series, supporting rational series for now 

## Functionality

The program creates an ast from tokenized input and analyzes said ast for the polynomial degree performing L'Hopitals until a result is reached.
